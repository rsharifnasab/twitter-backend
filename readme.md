## Getting started

1. Clone this repository:

```bash
git clone https://github.com/rsharifnasab/twitter-backend.git
```

2. build 

```bash
go build
```

3. run

```bash
./twitter-backend
```

4. api documentation with swagger
```
http://localhost:8585/swagger/index.html
```
